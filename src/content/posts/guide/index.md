Tutorial Dibawah ini adalah cara bagaimana caranya buat run Sc Buatan Gw, bagi yang gabisa aja Kalau Bisa mah ywdh gausah diliat wkkw

## Tutorial Run Panel 

```yaml
---
startup: node main.js --pairing-code --autocleartmp --autoread
caranya: Habis Startup Di Ganti Langsung Start aja
---
```

| Attribute     | Description                                                                                                                                                                                                 |
|---------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `startup`       | Buat Ngubah Method Bot Pairing/Scan                                                                                                                                                                                      |
| `File`   | Sudah wajib ada filenya lah kalau kgk ad gimana mau run.                                                                                                                                                                            |
| `settings` | Settings Bot Lihàt aja di Folder System Trs file settings.                                                                                                                                                   |
| `Foto`       | Buat Ubah Foto Dll Çari aja di settings.js Cari Yang telegraph|
| `Plugins`        | Buat Ubah Wm/Menu Ada di setiap Plug                                                                                                                                                    |
